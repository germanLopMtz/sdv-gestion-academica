# Maestros

Guia del modulo de gestion de maestros/profesores.

## Descripcion

Modulo para administrar la informacion de los profesores y su asignacion a materias.

## Endpoints principales

### Listar maestros

```http
GET /api/Maestro
```

### Crear maestro

```http
POST /api/Maestro
```

### Actualizar maestro

```http
PUT /api/Maestro/{id}
```

### Eliminar maestro

```http
DELETE /api/Maestro/{id}
```
